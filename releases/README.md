# PackDroid - File Release

## 📦 Versi Terbaru: v1.0.0

### File APK
[⬇️ Download APK](./app-release.apk) - Untuk install langsung di HP

### File AAB
[⬇️ Download AAB](./app-release.aab) - App Bundle untuk Play Store

### Cara Install APK
1. Download file APK
2. Transfer ke HP Android
3. Buka file APK dan install
4. Jika muncul peringatan, izinkan install dari sumber tidak dikenal

### Catatan
- Minimum Android: 8.0 (API 26)
- Ukuran: ~18 MB
- Lisensi: MIT
